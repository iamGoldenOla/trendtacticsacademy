# Conversational Course Creation Demo

This directory contains demonstration files for the conversational course creation feature powered by Trendy AI.

## Files

- `conversationalCourseDemo.js` - Contains sample conversation data and demo functions
- `ConversationalDemo.js` - React component that visualizes the conversation flow
- `ConversationalCourseTest.js` - Interactive test page for the feature

## Available Routes

1. `/course-creator` - Main course creation interface
2. `/conversational-course-test` - Interactive test page
3. `/conversational-demo` - Visualization demo of conversation flow

## How to Use

1. Navigate to `/conversational-demo` to see a visualization of how the conversation works
2. Navigate to `/conversational-course-test` to interact with a test version of the feature
3. Use `/course-creator` for the full production-ready course creation experience

## Features Demonstrated

- Natural language conversation flow
- Course content generation
- Interactive UI components
- Real-time conversation display
- Sample prompts for inspiration
