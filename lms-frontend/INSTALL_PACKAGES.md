# Install Required Packages

Run these commands in the `lms-frontend` directory:

```bash
cd lms-frontend

# Install animation and UI libraries
npm install framer-motion lucide-react recharts @monaco-editor/react react-markdown remark-gfm react-syntax-highlighter

# Install types
npm install --save-dev @types/react-syntax-highlighter
```

This will install:
- **framer-motion** - Smooth animations and transitions
- **lucide-react** - Professional, clean icon library
- **recharts** - Data visualization for progress charts
- **@monaco-editor/react** - Code editor for playground
- **react-markdown** - Render lesson content
- **remark-gfm** - GitHub Flavored Markdown support
- **react-syntax-highlighter** - Code syntax highlighting
