# Interactive Features Quick Start Guide

## How to Access Whiteboard and Playground

### Step 1: Navigate to the Vibe Coding Course
- Go to https://academy.trendtacticsdigital.com
- Find and click on the "Vibe Coding" course card
- Or click on "Courses" in the navigation menu and select "Vibe Coding"

### Step 2: Enroll in the Course
- On the course detail page, click the **"Enroll Now"** button
- The Vibe Coding course is free, so you'll be enrolled immediately
- You'll see a success message confirming your enrollment

### Step 3: Access the Learning Board
- After enrollment, click **"Continue Learning"** or **"Go to Course"**
- This takes you to the interactive course environment
- Click on the **"Learning Board"** tab at the top of the page

### Step 4: Use Interactive Features
Once on the Learning Board, you'll see:

#### Lesson Reading Board
- Interactive whiteboard for following along with lessons
- Content displays for each lesson in the course

#### Interactive Playground (at the bottom)
- **Code Editor Tab**: Write and test code directly in your browser
- **AI Prompt Playground Tab**: Interact with AI to get coding help and suggestions
- **Output Tab**: See results of your code execution

## Troubleshooting Tips

### If you don't see the interactive features:
1. Make sure you're enrolled in the course (click "Enroll Now" if needed)
2. Navigate to the "Learning Board" tab in the course interface
3. Check that JavaScript is enabled in your browser
4. Refresh the page if features don't load
5. Clear browser cache if experiencing loading issues

### If you're still having issues:
- Try using a different browser
- Make sure you're using the latest version of your browser
- Disable browser extensions that might interfere with the site
- Contact support if problems persist

## Course Structure
The Vibe Coding course contains 5 modules with 3 lessons each:
1. Understanding Vibe Coding
2. Thinking Before Coding
3. Working with AI as a Creative Partner
4. Vibe Coding in Real Life
5. Exploring and Building with Confidence

Each lesson can be accessed through the navigation panel on the left side of the Learning Board.
