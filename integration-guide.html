Integration Guide

Connecting HTML Course Content with React Dashboard Features

Hybrid Implementation Strategy

1

Course Content Structure

HTML course pages are organized in a clear directory structure that mirrors your course hierarchy:

courses/
├── index.html (Courses listing)
└── vibe-coding/
    ├── index.html (Course overview)
    ├── module-1-understanding-vibe-coding.html
    ├── module-2-thinking-before-coding.html
    ├── module-3-working-with-ai.html
    ├── module-4-vibe-coding-in-real-life.html
    └── module-5-exploring-with-confidence.html

Benefits of HTML Structure

  • Fast Loading: No JavaScript framework overhead
  • SEO Friendly: Search engines can easily crawl content
  • Reliable: No rendering failures or dependency issues
  • Easy Updates: Simple HTML editing without builds
2

Shared Authentication System

Both HTML and React components share the same authentication system using localStorage or cookies:

// In HTML pages - Check authentication status
<script>
function checkAuth() {
  if (localStorage.getItem('userToken')) {
    // User is logged in
    document.getElementById('auth-buttons').style.display = 'none';
    document.getElementById('user-menu').style.display = 'block';
  }
}
</script>

This ensures seamless navigation between HTML course content and React-powered dashboard features.

3

Navigation Integration

Unified navigation across both HTML and React sections:

// Common navigation component used in both HTML and React
<nav class="main-navigation">
  <a href="/">Home</a>
  <a href="/courses">Courses</a>
  <a href="/dashboard">Dashboard</a>
  <a href="/playground">Playground</a>
  <a href="/whiteboard">Whiteboard</a>
</nav>

This ensures users experience consistent navigation regardless of which section they're in.

4

Progress Tracking Integration

HTML course pages can communicate progress back to your React dashboard:

// Track lesson completion in HTML pages
<script>
function markLessonComplete(lessonId) {
  // Send completion status to backend
  fetch('/api/progress', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': 'Bearer ' + localStorage.getItem('userToken')
    },
    body: JSON.stringify({ lessonId, completed: true })
  });
  
  // Update UI
  document.getElementById('completion-' + lessonId).innerHTML = '✓ Completed';
}
</script>
